Title: Novel Hybrid Biometric Electronic Voting System

Description: In the current voting system Polling officer uses semipermanent ink for identifying the fraudulent voters. These methods require additional manpower. To overcome these problems, there is a new technique, which is based on Internet of Things (IOT). In this technique, the user has to enter the party ID using keypad module. Then the party icon will display on the monitor. The vote count of that party will be incremented and the total vote count of every party will be updated to the server. The system will have a biometric recognition to identify the fingerprint and face of the voter. The server used in this project is thingsspeak.com it reduces the manpower, The fraudulent voters can be identified automatically and the result would be hundred percent genuine. The advantages are Highly secured,Server can be accessed only by the admin and Immediate results can be shown on the server.
